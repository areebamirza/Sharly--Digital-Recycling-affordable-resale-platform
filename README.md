# Sharly – Digital Recycling & Affordable Resale Marketplace

Sharly is a MERN stack-based web application designed to promote sustainability by enabling users to donate unused items, support recycling, and purchase affordable reusable products using a point-based reward system.

---

## Features

* Donate unused items for recycling and reuse
* Earn reward points for each donation
* Redeem points for eco-friendly products
* Affordable resale marketplace for reusable items
* Secure authentication using JWT
* User dashboard for tracking donations and reward points
* Responsive user interface for multiple devices

---

## Tech Stack

**Frontend:**

* React.js
* React Router

**Backend:**

* Node.js
* Express.js
* JWT Authentication

**Database:**

* MongoDB
* Mongoose

---

## Project Objective

Sharly aims to reduce waste and promote sustainable consumption by connecting users for donation, recycling, and affordable resale through a reward-based system.
